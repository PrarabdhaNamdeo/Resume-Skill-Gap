# Resume Skill Gap Analyzer (ML Project)

This project is a Machine Learning-based Resume Skill Gap Analyzer that compares a candidate's resume with a job description and identifies missing skills required for the role.

It helps job seekers understand what skills they need to improve in order to match industry expectations.

## Features

- Extracts skills from resume text
- Extracts skills from job description
- Compares both skill sets
- Identifies missing skills
- Suggests improvement areas
- Simple and easy-to-use interface
- Supports multiple job roles

## Tech Stack

- Python
- Scikit-learn
- Pandas
- NLP (TF-IDF / Keyword Matching)
- Streamlit (for UI)

## How It Works

1. User uploads resume
2. User enters job description
3. System extracts important skills using NLP
4. Model compares resume vs job requirements
5. Missing skills are displayed as output

## Use Case

This project helps:

- Students preparing for placements
- Job seekers improving resumes
- Career switchers identifying skill gaps
- Recruiters screening candidates
